# Gift Card Billing System

This project is a console-based application developed using Java. 
It helps manage gift card billing operations.

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Features
- Add gift card details
- View gift card records
- Update gift card information
- Delete gift card records

## Database
MySQL database is used to store gift card and billing details.
